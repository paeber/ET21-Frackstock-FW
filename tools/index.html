Firmware

Die Firmware ist in arbeit. Über den Konfigurator können Einstellungen auf dem Frackstock angepasst werden. Jeder Frackstock hat eine Definierte ID im Flash. Sollte die ID 0x00 oder 0xFF sein funktioniert das Ganze nicht. Bitte bei Eberkreis melden!
