LED-Ring

Erweiterungsplatine für das Baseboard. Ein LED-Ring mit 12 WS2812 RGB LEDs umrahmt zwei 7-Segment anzeigen.
Positionen R21, R22 und R24 werden nicht Bestückt.
Positionen C17, C18, C21 und C22 werden nicht Bestückt.

Firmware

Die Firmware basiert auf dem Pico C SDK. Über den Konfigurator können Einstellungen auf dem Frackstock angepasst werden. Jeder Frackstock hat eine Definierte ID im Flash. Sollte die ID 0x00 oder 0xFF sein funktioniert das Ganze nicht. Bitte bei Eberkreis melden!

  1. Verbinden über USB C mit dem PC
  2. Laden Sie die neueste Firmware von den GitHub Releases herunter
  3. Verbinden Sie sich mit dem Gerät im Konfigurator
  4. Drücken Sie "Flash Mode", um das Gerät am PC zu mounten
  5. Kopieren Sie Frackstock.uf2 auf das Gerät
